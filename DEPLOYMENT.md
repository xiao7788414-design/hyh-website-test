# 华与华官网上线说明

本发布包包含中英文网站、英文 NEWS 页面、全部正式素材，以及“联系我们”表单的邮件发送服务。

## 运行要求

- Node.js 20 LTS 或更高版本
- 服务器能够访问 `smtp.exmail.qq.com:465`
- 网站目录对运行账号可读，`data/` 目录可写
- 正式环境必须以 Node 服务运行，不能只上传到纯静态空间

## 安装与启动

1. 解压发布包并进入网站目录。
2. 安装生产依赖：

   ```bash
   npm ci --omit=dev
   ```

3. 检查 `.env` 中的正式域名、SMTP 发件账号和收件账号。
4. 启动邮件服务：

   ```bash
   npm run start:email
   ```

建议使用服务器进程管理工具保持服务常驻，并将 Web 服务器反向代理到 `.env` 中 `PORT` 对应的本机端口。

## 反向代理

- 通过 Nginx、宝塔或云平台反向代理时，将 `TRUST_PROXY` 设为 `true`。
- 将 `SITE_ORIGIN` 改为正式官网完整来源，例如 `https://www.huayuhua.com`。
- 代理层请求体上限应大于 10MB，建议设置为 12MB 或更高，否则表单附件会在到达网站服务前被拦截。

## 上线验收

1. 检查中文首页、英文首页、英文 NEWS 列表及新闻详情页。
2. 检查中英文切换和所有主导航入口。
3. 在“联系我们 / 业务合作”提交一次真实测试，确认页面提示成功且收件邮箱收到邮件。
4. 分别测试无附件和带附件提交。
5. 确认服务器上的 `data/contact-submissions.jsonl` 能记录提交结果。

## 安全说明

本次私有测试包按要求包含真实 `.env`。其中含邮箱授权凭据，必须限制文件读取权限，不得上传至公开代码仓库或通过公开链接分发。上线完成后如怀疑压缩包曾泄露，应立即更换邮箱授权码。
